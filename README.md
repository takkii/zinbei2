## zinbei2

＜:ロミ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

WindowsではRubyGemsで配布している試用版を使います。

有償版zinbei(2.0.1)のエンコーディングは、utf8です。

OSS版は、限定的機能でかつ小分けプロジェクトで開発しています。

さらにUNIXに向けて進化したzinbei2です。

ご利用は可能ですが、継続利用のときなどはライセンスの購入をお待ちしています。どうぞご検討願います。

[Vector zinbei2 ライセンスの購入先](https://www.vector.co.jp/soft/data/prog/se516665.html?_ga=2.75685973.1169120073.1527936228-1534765938.1527936228)

※ 購入するとzipファイルを開くパスワードがVectorから送付するはずです。そのキーがzinbei2または箱入り娘プロジェクトのライセンスですので、大事に保管してください。

＜:ロミ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

試用版

```ruby
gem install zinbei
```

有償版

```ruby
brew tap takkii/homebrew-zinbei2
brew update
brew install zinbei2
```

※ OSS版のzinbeiに上書きしても動作します。
